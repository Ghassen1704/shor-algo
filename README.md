# Shor Algorithm Experiments

Expérimentations de l'algorithme de Shor pour différents entiers N.

## Fichiers

- `shor_N_15.ipynb` : Shor pour N=15  
- `shor_N_5.ipynb`  : Shor pour N=5  
- `shor_N_21.ipynb` : Shor pour N=21  
- `addition_modulaire.ipynb` : fonctions utilitaires pour les calculs modulaires

## Usage

Ouvrir le notebook correspondant à l'entier N et exécuter les cellules.  

> Ces implémentations sont des simulations pédagogiques et ne tournent pas sur un vrai ordinateur quantique.
